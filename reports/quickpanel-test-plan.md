# Quick Panel UI 验收测试计划

## 背景

Focus Copilot V3.0 对快捷面板进行了重大重构（移除 Pin 概念、新增 Tab/高亮/启动功能），并在后续修复中移除了底部按钮栏。需要对面板所有功能进行 UI 验收，发现问题即修复。

## 测试前置条件

1. `make install` 安装最新版本
2. 授权辅助功能权限：系统设置 → 隐私与安全性 → 辅助功能 → Focus Copilot 开启
3. 在主看板中配置 3+ 个 App（含至少 1 个多窗口 App、1 个未运行 App、1 个收藏 App）

---

## 测试用例

### A. 面板弹出与收起（5 项）

| # | 测试项 | 操作步骤 | 预期结果 | 结果 |
|---|--------|---------|---------|------|
| A1 | Hover 弹出 | 鼠标悬停悬浮球 300ms | 面板从悬浮球方向滑出，150ms 动画，未钉住状态 | |
| A2 | Hover 收起 | A1 后鼠标离开悬浮球和面板 | 500ms 后面板自动淡出收起 | |
| A3 | 单击弹出+钉住 | 面板未显示时单击悬浮球 | 面板弹出且自动钉住（图钉变红），鼠标离开不收起 | |
| A4 | 单击取消钉住 | A3 后再次单击悬浮球 | 面板取消钉住并关闭 | |
| A5 | Hover 可见时单击 | hover 弹出面板后立即单击悬浮球 | 面板不闪烁，直接变为钉住状态 | |
