# FocusPilot 图标设计解析：禅圆（Enso）

## 图标构成

图标由两个核心元素叠加而成：

1. **底层：渐变球体** — 蓝色径向渐变圆球，带有左上高光和底部暗影，营造出立体的「悬浮球」质感
2. **上层：禅圆 Enso** — 一条白色未闭合弧线（300 度），从起笔墨滴开始，笔触由粗渐细，尾部以三个渐隐散点收束

## 与 FocusPilot 的寓意关联

### 1. 禅圆 = 专注的象征

Enso（円相）是禅宗书法中最具代表性的符号，僧人在一次呼吸内一笔画成。它代表的核心理念恰好是 FocusPilot 的产品哲学：

- **专注当下**：一笔挥就，不可重来，正如番茄钟工作法要求你沉浸在当前时段
- **不追求完美**：弧线故意不闭合（300 度而非 360 度），留有 60 度的缺口——专注不是苛求完美，而是在有限时间内做到最好

### 2. 未闭合的弧 = 循环但不封闭

FocusPilot 的核心功能是「工作 -> 休息 -> 工作」的番茄钟循环。禅圆的不闭合设计暗示：

- 每一轮专注都有起点和终点（起笔墨滴 -> 收笔散点）
- 但它不是封闭的死循环，而是**螺旋上升**的——每次休息后的新一轮都是新的开始

### 3. 笔触从粗到细 = 能量的节奏

弧线的绘制参数很讲究：

- `maxWidth = size * 0.09`（起笔粗）-> `minWidth = size * 0.02`（收笔细）
- 透明度从 `0.88` 渐降至 `0.73`
- 使用 `easedT = 1.0 - pow(1.0 - t, 2.0)` 缓出曲线

这模拟了毛笔书写的自然力度变化，也隐喻了**专注时段中能量的自然消耗**——从全神贯注到柔和收尾，而非戛然而止。

### 4. 起笔墨滴 = 启动的决心

起始处的圆形墨滴（`inkDropRadius = size * 0.07`），饱满而坚定，象征每次点击「开始专注」时的**果断投入**。

### 5. 收笔散点 = 优雅的过渡

尾部三个渐隐散点（半径递减 `0.042 -> 0.030 -> 0.020`，透明度递减 `0.65 -> 0.42 -> 0.22`）不是突然截断，而是如墨迹自然洇散。这对应 FocusPilot 中工作阶段结束后**不是硬切，而是有弹窗提示、有 pending 状态**的柔和过渡设计。

### 6. 悬浮球体 = 常驻陪伴

底层的立体球体正是 app 在桌面上的存在形态——一个**始终悬浮在屏幕边缘的小球**，随时待命。径向渐变和高光赋予它真实的物理感，暗示它不是一个冰冷的工具，而是一个有温度的「领航员」（Pilot）。

## 一句话总结

> **一笔未闭合的禅圆，画在一颗悬浮的蓝色球体上** —— 用禅宗"一期一会"的专注哲学，承载一个桌面效率工具的产品灵魂。

## 技术实现

- 图标生成脚本：`scripts/gen-icon.swift`
- 悬浮球绘制：`FocusPilot/FloatingBall/FloatingBallView.swift` 中的 `createBrandLogoImage` 方法
- 两处绘制逻辑保持一致，确保悬浮球与 App 图标视觉统一
