## 布局结构

类似 VS Code 布局，由活动栏 + 侧边栏 + 工作区组成。

```
┌─ 活动栏 ─┬─ 侧边栏 ──────┬─ 工作区 ─────────────────────────┐
│  52px    │  260px        │                                    │
│          │               │  顶部栏（标题 + 视图切换 + 筛选）    │
│  🏠 Home │  随活动栏      │  ─────────────────────────────     │
│  📥 Inbox│  切换内容      │                                    │
│  🎯 Focus│               │  内容区（随页面和视图变化）           │
│  💻Studio│               │                                    │
│  📁 Areas│               │                                    │
│  🤖 Crew │               │                                    │
│  ─ ─ ─   │               │                                    │
│  ⚙️ Set  │               │                                    │
└──────────┴───────────────┴────────────────────────────────────┘
```

## 活动栏

左侧固定 52px，纵向排列图标。点击切换对应的侧边栏和工作区内容，选中项高亮显示 + 左侧蓝色指示条。

| 图标 | 页面 | 说明 | 设计文档 |
|------|------|------|---------|
| 🏠 | Home | AI 对话入口 + 最近操作 | [01-home.md](01-home.md) |
| 📥 | Inbox | 收集箱：待阅读/灵感/临时任务/便签 | [02-inbox.md](02-inbox.md) |
| 🎯 | Focus | 结构化行动工作台：规划+看板+列表+Task 执行闭环 | [03-workspace.md](03-workspace.md) |
| 💻 | Studio | 项目级 AI 工作区：对话/文件浏览/Git 历史 | [04-studio.md](04-studio.md) |
| 📁 | AreaProjects | 项目资产沉淀（执行类+知识类） | [05-area-projects.md](05-area-projects.md) |
| 🤖 | AICrew | Agent 团队管理 | [06-ai-crew.md](06-ai-crew.md) |
| ⚙️ | Settings | 全局配置（底部） | [07-settings.md](07-settings.md) |

### Focus vs Studio vs AreaProjects

| | Focus 🎯 | Studio 💻 | AreaProjects 📁 |
|---|---|---|---|
| 关注点 | 计划和任务怎么推进 | 和 Agent 怎么工作 | 资料和产物沉淀在哪里 |
| 核心对象 | WorkItem（目标/Task） | CodeSession（会话） | 文件/文件夹 |
| 交互模式 | 看板拖拽/列表/甘特 | 对话流/内联 Diff/审批 | 文件树/Markdown 编辑 |

## 侧边栏

宽度 260px，随活动栏选择切换内容。通常包含：
- 顶部标题或 Tab 条（Studio 使用顶部 Tab 条：🤖 会话 / 📁 文件 / 🌿 Git）
- 搜索框
- 分类/导航列表

## 工作区

主内容区域，flex: 1 自适应宽度。通常包含：
- 顶部栏：页面标题 + 视图切换 Tab + 筛选按钮 + 操作按钮
- 内容区：随页面和视图变化

点击顶部标题栏最右边可收起/展开右半工作区（部分页面支持双栏）。
