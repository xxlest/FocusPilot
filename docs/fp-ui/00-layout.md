## 布局结构

类似 VS Code 布局，由活动栏 + 侧边栏 + 工作区组成。Studio 页面额外支持右侧栏（文件浏览）和终端面板。

### 基础布局（Home / Projects / Studio / Review / AICrew / Settings）

```
┌─ 活动栏 ─┬─ 侧边栏 ──────┬─ 工作区 ─────────────────────────┐
│  52px    │  260px        │                                    │
│          │               │  顶部栏（标题 + 视图切换 + 筛选）    │
│  🏠 Home │  随活动栏      │  ─────────────────────────────     │
│  📂 Proj │  切换内容      │                                    │
│  💻Studio│               │  内容区（随页面和视图变化）           │
│  🧠Review│               │                                    │
│  🤖 Crew │               │                                    │
│  ─ ─ ─   │               │                                    │
│  ⚙️ Set  │               │                                    │
└──────────┴───────────────┴────────────────────────────────────┘
```

### Studio 扩展布局（含右侧栏 + 终端面板）

```
┌─ 活动栏 ─┬─ 侧边栏 ──────┬─ 工作区 ──────────────┬─ 右侧栏（可选）─┐
│  52px    │  260px        │                        │  ⌘⇧B 切换       │
│          │               │  顶栏（项目·Runtime·   │                  │
│  🏠 Home │  全局视图       │   🔧工具·🖥终端·◨侧栏）│  多 Tab 文件浏览  │
│  📂 Proj │  +项目列表      │  ────────────────     │  只读预览        │
│  💻Studio│  搜索           │                        │                  │
│  🧠Review│  (统一侧边栏)  │  对话区（消息流）       │                  │
│  🤖 Crew │               │                        │                  │
│  ─ ─ ─   │               │  输入区（底部）         │                  │
│  ⚙️ Set  │  [⚙ 设置]     │  终端面板（可选，⌘J）  │                  │
└──────────┴───────────────┴────────────────────────┴──────────────────┘
```

Studio 独有区域：右侧栏（⌘⇧B 切换，多 Tab 文件浏览）、终端面板（⌘J 切换，输入区下方最底部，多实例 Tab）。详见 [04-studio.md](04-studio.md)。

## 活动栏

左侧固定 52px，纵向排列图标。点击切换对应的侧边栏和工作区内容，选中项高亮显示 + 左侧蓝色指示条。

| 图标 | 页面 | 说明 | 设计文档 |
|------|------|------|---------|
| 🏠 | Home | 全局概览入口：摘要数字条 + 重点列表 + 对话视图（Tab 切换） | [01-home.md](01-home.md) |
| 📂 | Projects | 记忆与输入管理层：Inbox（收集）+ Projects（知识沉淀），侧边栏 Tab 切换 | [05-area-projects.md](05-area-projects.md) |
| 💻 | Studio | 跨项目 AI 工作台：任务视图（默认看板 + 泳道/列表/时间轴视图）+ 项目视图（工作项/对话/Diff/终端） | [04-studio.md](04-studio.md) |
| 🧠 | Review | 复习与内化中心：今日复习 / 内化挑战 / 卡片库 / 统计 | [06-review.md](06-review.md) |
| 🤖 | AICrew | Agent 团队管理 | [07-ai-crew.md](07-ai-crew.md) |
| ⚙️ | Settings | 全局配置（底部） | [08-settings.md](08-settings.md) |

### Studio vs Projects vs Review

| | Studio 💻 | Projects 📂 | Review 🧠 |
|---|---|---|---|
| 关注点 | 任务怎么推进、和 Agent 怎么工作 | 信息在哪里、归到哪里、沉淀在哪里 | 知识如何被记住并内化 |
| 核心对象 | WorkItem + Session + ExecutionRun | InboxItem + 文件/文件夹 | ReviewCard（卡片） |
| 交互模式 | 看板/泳道/列表/时间轴展示 + 对话/Diff/Git/终端 | Inbox 列表 / 文件树 / Markdown 编辑 | 复习队列/内化挑战/统计 |

## 侧边栏

宽度 260px，⌘B 收起/展开，随活动栏选择切换内容。通常包含：
- 顶部标题或 Tab 条
- 搜索框
- 分类/导航列表

**Projects 侧边栏特殊结构**：顶部 `[📥 Inbox | 📂 Projects]` Tab 切换。Inbox Tab 显示分类导航（待阅读/灵感/临时事项）；Projects Tab 显示文件树。Tab 上 Inbox 角标显示未处理数。详见 [05-area-projects.md](05-area-projects.md)。

**Studio 侧边栏特殊结构**：顶部 `[任务视图 | 项目视图]` 切换。任务视图显示范围筛选（今日/本周/本月/全局/执行中/等我决策/Triage）；项目视图显示 Workspace 项目列表（每个项目含工作项/对话/自动化子节点）→ 全局自动化（可折叠）→ 底部设置。详见 [04-studio.md](04-studio.md) §6。

**Review 侧边栏特殊结构**：固定为四个一级入口：`今日复习 / 内化挑战 / 卡片库 / 统计`。底部展示连续复习天数、弱项卡数量、关键卡待确认数。

## 工作区

主内容区域，flex: 1 自适应宽度。通常包含：
- 顶部栏：页面标题 + 视图切换 Tab + 筛选按钮 + 操作按钮
- 内容区：随页面和视图变化

**Studio 工作区扩展**：任务视图选中时，侧边栏决定计划范围，工作区默认显示 `▦ 看板`，并通过单一视图下拉切换显示模式；下拉菜单上方为 `执行视图`（`▦ 看板 / 田 泳道 / ☰ 列表`），下方为 `规划视图`（`▱ 时间轴`）。顶部 `时间 / 目标 / 来源` 作为当前范围内的二级筛选同步作用于时间轴、看板、列表和泳道；泳道模式额外显示 `▤ Workspace / ◉ 执行 Agent` 分组控件，可在 Workspace × 状态和 Agent × 状态之间切换，列表和看板不显示该按钮；看板卡片可拖到其他状态列。项目选中时显示工作项/对话 Tab；对话打开时工作区右侧可展开右面板（⌘⇧B，任务/Diff/Git 三 Tab），输入区下方可展开终端面板（⌘J）。详见 [04-studio.md](04-studio.md) §7。

**Review 工作区结构**：`今日复习` 以单卡队列为主，`内化挑战` 以题面 + 回答区为主，`卡片库` 以列表 + 详情为主，`统计` 以趋势和分布图为主。
