# Settings 页面设计

> **状态**：草稿
> **更新**：2026-05-15

---

## 1. 定位

Settings 是全局配置中心，管理 FocusPilot 各模块的偏好设置和默认行为。

### 与 AICrew 的职责边界

| 维度 | Settings | AICrew |
|------|----------|--------|
| 管理对象 | 全局偏好和默认值 | Agent 成员的详细配置 |
| Agent 相关 | 引用默认 Agent、配置并发上限 | 管理每个 Agent 的角色/能力/MCP |

**原则**：Settings 管全局默认值和偏好，具体 Agent 配置去 AICrew。

## 4. 核心交互

| 用户任务 | 操作路径 |
|---------|---------|
| 修改默认 Agent | 侧边栏 Focus -> 执行引擎 -> 默认 Agent 下拉选择 |
| 切换主题 | 侧边栏 外观 -> 主题下拉选择 -> 即时预览 |
| 自定义快捷键 | 侧边栏 快捷键 -> 点击快捷键项 -> 录入新组合键 |
| 配置项目模板 | 侧边栏 Projects -> 项目模式/模板配置 |

---

## 5. 状态与规则

### 配置分区

| 分区 | 配置内容 |
|------|---------|
| Focus | 规划引擎（默认 Skill、自动规划）、执行引擎（默认模式、默认 Agent、并发上限） |
| Studio | Runtime 偏好、文件浏览默认行为 |
| AICrew | Agent 列表入口（跳转到 AICrew 页面） |
| Projects | 项目模式默认值（Agile/Flow/Lite/Free）、项目模板 |
| 外观 | 主题选择、悬浮球外观 |
| 快捷键 | 全局快捷键配置 |
| 通用 | 启动行为、通知偏好、数据目录 |

### 保存行为

- 所有配置修改即时生效，无需手动保存
- 主题切换即时预览

## 7. 术语

| 术语 | 含义 |
|------|------|
| Settings | 页面名称（活动栏 ⚙️） |
| 规划引擎 | Focus 页面中 AI 辅助规划的配置 |
| 执行引擎 | Focus 页面中 Task 执行的配置 |
| 项目模式 | 项目管理风格（Agile/Flow/Lite/Free） |
