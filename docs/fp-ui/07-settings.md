# Settings 页面设计

> **状态**：草稿
> **更新**：2026-04-18

---

## 定位

Settings 页面提供全局配置能力。

## 配置分区

### Workspace 配置

- **规划引擎**
  - 默认 Skill（如 brainstorming）
  - 可用 Skill 列表
  - 自动规划开关
- **执行引擎**
  - 默认执行模式（auto / manual）
  - 默认 Agent
  - Agent 并发数
- **Agent 列表**（引用 AICrew 配置）

### AICrew 配置

- Agent 角色管理（详见 [06-ai-crew.md](06-ai-crew.md)）

### Projects 配置

- 项目模式默认值（Agile/Flow/Lite/Free）
- 项目模板

### 通用设置

- 快捷键
- 主题选择
- 悬浮球外观
- 其他偏好
