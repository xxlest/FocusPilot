> **状态**：可开发
> **更新**：2026-05-22
> **参考**：Plane Home 页（Widget 化布局）、Claude.ai / ChatGPT（对话首页）、Codex Desktop（任务工作台）
> **关联**：[PRD §3.6 Dashboard 驾驶舱](../PRD.md)、[03-focus.md](03-focus.md)

---

## 1. 定位

Home 是 FocusPilot 的**全局概览入口**，提供数据摘要和对话入口。用户打开主面板后的默认落地页。

### 与 Focus 的职责边界（硬规则）

| 维度 | Home | Focus |
|------|------|-------|
| 定位 | 摘要 + 跳转 + 恢复上下文 | 规划 + 执行 + 评估 + 验收 |
| 数据展示 | 聚合计数 + 摘要列表（前 N 条） | 完整列表 + 筛选 + 排序 + 分组 |
| 允许操作 | 打开、跳转、恢复上下文 | 新建、拖拽、改状态、批量处理、详情编辑 |
| 不允许操作 | 新建、拖拽、改状态、批量处理 | -- |

**原则**：Home 是导航层，不是第二个 Focus。看到数据、点进去操作。

## 3. 工作区

工作区有两个视图：**概览视图**（默认）和**对话视图**，由侧边栏 Tab 和操作驱动切换。

### 3.1 概览视图（默认）

顶部摘要数字条 + 下方重点列表。

#### 摘要数字条

```
┌──────┬──────┬──────┬──────┬──────┬──────┐
│  4   │  12  │  28  │  45  │  5   │  3   │
│ 今日 │ 本周 │ 本月 │ TODO │ 项目 │ 执行 │
└──────┴──────┴──────┴──────┴──────┴──────┘
```

6 个指标并排，每个可点击跳转到对应页面（Focus / AreaProjects）。

#### 重点列表

数字条下方展示两个最需要关注的列表：

**🔄 执行中**

```
┌─ 🔄 执行中 ─────────────────────────────┐
│ 客户画像生成      ████░░ 60%   大客户    │
│ API 文档更新      ██░░░░ 30%   FocusPilot│
│ 周报自动汇总      █░░░░░ 10%   内部      │
└──────────────────────────────────────────┘
```

- 展示所有 `status = in_progress` 的 WorkItem（通常数量不多）
- 每行：任务名 + 进度条 + 归属项目
- 点击行 -> 跳转到 Focus 对应 Task 详情

**🔥 今日待办**

```
┌─ 🔥 今日待办 (4/7) ─────────────────────┐
│ ☐ frontmatter 解析           FocusPilot  │
│ ☐ 审阅 PR #18               FocusPilot  │
│ ☐ 客户跟进电话               大客户      │
│                 查看全部 ->              │
└──────────────────────────────────────────┘
```

- 展示前 N 条（默认 5 条），超出显示"查看全部 ->"
- 每行：状态图标 + 任务名 + 归属项目
- 点击行 -> 跳转到 Focus 对应 Task 详情
- "查看全部 ->" -> 跳转到 Focus 看板，筛选今日

### 3.2 对话视图

用户在侧边栏"💬 对话"Tab 中点击对话条目或"新建对话"时，工作区从概览切换为对话视图。

#### 新建对话（空态）

```
┌─ 工作区 ─────────────────────────────────┐
│                                           │
│              🤖                           │
│          FocusPilot                       │
│      有什么可以帮你？                      │
│                                           │
│  [🤖 通用 ▾] ┌─────────────────────────┐ │
│              │ 描述你想做的事... [↑ 发送]│ │
│              └─────────────────────────┘ │
│                                           │
└───────────────────────────────────────────┘
```

- 居中问候语 + Agent 选择器 + 输入框
- **Agent 选择器**：输入框左侧下拉控件，默认「🤖 通用」，展开显示 Crew 成员列表
- 发送后进入对话消息流，所选 Agent 锁定到该对话（不可中途切换）

#### 对话消息流

```
┌─ 工作区 ─────────────────────────────────┐
│ 市场分析报告                 [🤖 研究员 🔒]│  <- 顶栏：标题 + 锁定 Agent
│ ─────────────────────────────────────── │
│                                           │
│  👤 帮我分析一下 Q2 的客户数据             │
│                                           │
│  🤖 好的，我来分析 Q2 客户数据。          │
│     以下是主要发现：                       │
│     1. 新增客户 23 个，同比增长 15%        │
│     2. ...                                │
│                                           │
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  │
│  [🤖 研究员 🔒] ┌─────────────────────┐  │
│                 │ 继续提问...  [↑ 发送]│  │
│                 └─────────────────────┘  │
└───────────────────────────────────────────┘
```

- **顶栏**：左侧对话标题（Agent 自动生成，可右键重命名），右侧 Agent pill（只读，显示锁定状态）
- **消息区**：用户消息 + Agent 回复，Markdown 渲染，支持代码块语法高亮
- **输入区**：多行自适应高度（1~6 行），Enter 发送，Shift+Enter 换行；左侧 Agent 标识（锁定态，不可切换）
- **对话标题**：首次发送后由 Agent 自动生成简短标题，显示在顶栏和侧边栏
- 点击侧边栏其他对话 -> 切换对话上下文
- 点击侧边栏"📊 概览"Tab -> 工作区回到概览视图

## 4. 核心交互

| 用户任务 | 操作路径 |
|---------|---------|
| 查看今日工作全貌 | 打开 Home -> 概览视图（默认） |
| 快速跳转到某个待办 | 概览视图点击任务行 -> 跳转 Focus 对应 Task |
| 开始一次自由对话 | 侧边栏"💬 对话"Tab -> [+ 新建对话] -> 选 Agent -> 输入发送 |
| 用指定 Agent 对话 | [+ 新建对话] -> Agent 下拉选择 Crew 成员 -> 输入发送 |
| 继续之前的对话 | 侧边栏"💬 对话"Tab -> 点击历史对话条目 |
| 从对话回到概览 | 侧边栏切到"📊 概览"Tab |
| 置顶/归档/分享对话 | 对话条目右键菜单 |
| 重命名对话 | 对话条目右键菜单 或 顶栏标题右键 |

## 5. 状态与规则

### 视图切换规则

- 切换触发：
  - 侧边栏"💬 对话"Tab 中点击对话条目 -> 对话视图
  - 侧边栏"💬 对话"Tab 中点击"+ 新建对话" -> 对话视图（空态）
  - 侧边栏切到"📊 概览"Tab -> 概览视图
  - 从其他页面切回 Home -> 恢复上次离开时的视图状态

### Agent 选择规则

- 新建对话时通过输入框左侧下拉选择 Agent，默认「🤖 通用」
- **首次发送后 Agent 锁定**，对话内不可切换（对标 Studio 的 Runtime 锁定）
- 锁定后输入区和顶栏均显示 🔒 标识
- 想用不同 Agent 需新建对话

### 操作边界（硬规则）

Home 页**允许**的操作：
- 点击跳转到其他页面
- 新建/切换/继续对话
- 选择 Agent（新建对话时）
- 侧边栏 Tab 切换
- 对话条目：置顶、重命名、分享、归档、删除（右键菜单）

Home 页**不允许**的操作：
- 新建 Task / WorkItem
- 拖拽改状态或排序
- 批量操作
- 内联编辑任何字段

## 6. 数据对象

| 对象 | 说明 | 详细定义 |
|------|------|---------|
| WorkItem | 概览视图中展示的待办/目标/执行中任务 | [PRD §3.3](../PRD.md) |
| HomeConversation | 自由对话，无项目绑定 | 见下方会话模型 |

### 会话模型：Home 对话 vs Studio 对话

| 维度 | Home 对话 | Studio 对话 |
|------|----------|------------|
| 上下文 | 自由聊天，无项目绑定 | 绑定具体项目目录 |
| 用途 | 通用问答、规划建议、数据分析 | 项目级编码、文件操作、Git |
| Agent 选择 | 可选任意 Crew 成员 | 绑定项目配置的 Runtime |
| 历史归属 | Home 对话列表 | Studio 项目 Session 列表 |
| 返回行为 | 切回概览视图 | 切回 Studio 其他 Session |

---

## 7. 术语

| 术语 | 含义 |
|------|------|
| Home | 页面名称（活动栏 🏠） |
| 概览视图 | Home 工作区的默认视图（摘要数字条 + 重点列表） |
| 对话视图 | Home 工作区的对话视图（消息流 + 输入框） |
| 今日待办 | `scheduled_date = today` 的未完成 WorkItem |
| 本周待办 | `scheduled_date ∈ 本周` 的未完成 WorkItem |
| 本月目标 | `schedule = month` 的项目级目标（非普通待办） |
| Backlog | `schedule = backlog` 的待排期 WorkItem |

---

### 指标口径定义

| 指标 | 口径 | 跳转目标 |
|------|------|---------|
| 今日待办 | `scheduled_date = today` 且 `status ∉ {done, cancelled}` | Focus 看板，筛选今日 |
| 本周待办 | `scheduled_date ∈ 本周` 且 `status ∉ {done, cancelled}` | Focus 看板，筛选本周 |
| 本月目标 | `schedule = month` 的项目级目标 | Focus 列表，筛选本月 |
| 全部 TODO | 所有 `status ∉ {done, cancelled}` 的 WorkItem | Focus 列表，全量 |
| 活跃项目 | `lifecycle = active` 的项目 | AreaProjects |
| 执行中 | `status ∈ {in_progress, in_evaluation}` 的 WorkItem | Focus 看板，筛选执行中 |
