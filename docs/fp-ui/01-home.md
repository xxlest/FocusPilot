## 定位

Home 页面是 AIOS 的统一入口，提供 AI 对话框和最近操作记录。

## 核心功能

### AI 对话框

- 选择 AICrew 配置指定的 Agent
- 作为整个 AIOS 的唯一对话入口
- 后续移动端对话接入界面入口
- 实现项目和待办事项所有的对话内容

### Widget（参考 Plane Home 页）

- 最近操作记录
- 快捷链接
- 其他 Widget（待定义）
